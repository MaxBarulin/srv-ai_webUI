# Инструкции

Три пошаговые инструкции — выбери нужную:

| Инструкция | Для кого | Когда |
|------------|----------|-------|
| [Быстрый локальный тест](local-test-quick.md) | разработчик на Windows | быстро пощупать чат/заметки/календарь у себя на компьютере, без фильтра ПДн и PDF-сканов |
| [Полный локальный тест](local-test-full.md) | разработчик на Windows | проверить **все** функции локально: база знаний, фильтр ПДн (Наташа), PDF-сканы (poppler) |
| [Развёртывание на сервер](../deploy/README-deploy.md) | системный администратор (sudo) | боевая установка на сервере, где крутится llama.cpp; systemd, firewall, бэкап, офлайн-контур |

Порядок на практике: сначала «быстрый» или «полный» локальный тест на своём
компьютере, затем — «развёртывание на сервер».
